# Arduino-MCP

Arduino MCP Server - Model Context Protocol implementation for Arduino development

## 概要

このプロジェクトは、Arduino開発をAIアシスタントと統合するためのModel Context Protocol (MCP) サーバー実装です。

## 機能

- Arduinoボードとの通信
- スケッチのアップロード
- シリアルモニタリング
- センサーデータの取得

## セットアップ

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev
```
